WHAT WE DO

A five-step cycle, run on every node in the network.

Assess
Score priority nodes for investigation.
Monitor
Track interactions and change over time.
Manage
Intervene to restore broken edges.
Replicate
Scale the method to new sites.
Report
Publish open methods, code and data.
